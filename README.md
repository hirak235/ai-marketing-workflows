# AI Marketing Workflows

Real-world AI workflows for growth, performance marketing, and analytics.

This repository contains practical use cases, prompts, and system designs for using AI tools like Claude, ChatGPT, and others in real business environments.

---

## Why this repo exists

Most AI examples are generic. This repo focuses on actual workflows used in:

- Paid media (Google Ads, Meta Ads)
- Analytics (GA4, Looker Studio)
- CRM & lifecycle (HubSpot, Salesforce)
- GTM strategy and experimentation

These are based on real-world execution, not theory.

## Core Use Cases

### 1. Performance Marketing Optimization

**Prompt:**Act as a performance marketing expert. Analyze this campaign data and identify:

Wasted spend
Scaling opportunities
Audience inefficiencies
Creative gaps

Then provide a prioritized action plan with expected impact.
---

### 2. Landing Page Conversion Optimization

**Prompt:**You are a CRO expert. Review this landing page and identify:

Messaging gaps
UX friction points
CTA improvements
Trust signal weaknesses

Then rewrite the page to maximize conversion rate.
---

### 3. GTM Strategy Builder

**Prompt:**Act as a VP of Marketing. Build a go-to-market strategy for [product].

Include:

Target audience
Positioning
Channels
Budget allocation
KPIs
90-day execution roadmap
---

### 4. Marketing Analytics Debugging (GA4 / Tracking)

**Prompt:**Act as a marketing analytics engineer.

Help diagnose tracking issues in GA4:

Missing conversions
Attribution gaps
Event duplication
Funnel drop-offs

Provide step-by-step fixes using GTM and GA4.
---

### 5. AI-Powered Ad Copy Generation

**Prompt:**Generate high-converting ad copy for [product].

Include:

5 headlines
5 primary texts
Emotional + performance angles
A/B test variations
